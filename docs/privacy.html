隱私權政策

最後更新：2026 年 6 月

1. 收集的資料

本服務（中印雙向翻譯 LINE Bot）僅處理使用者透過 LINE 傳送的文字訊息，用於執行翻譯功能。 我們不會儲存任何訊息內容或個人識別資訊。

2. 資料使用方式

使用者傳送的文字會即時轉送至 Groq API 進行翻譯，翻譯完成後立即回傳，不會保留於伺服器。

3. 第三方服務

本服務使用以下第三方服務：

  • LINE Messaging API
  • Groq API

4. 聯絡方式

如有任何疑問，請透過 LINE 官方帳號與我們聯繫。
